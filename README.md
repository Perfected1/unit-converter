# Unit Converter (Go CLI)

A simple command-line unit converter built with Go.  
It supports conversions between **weight, distance, and volume units**.

---

## 🚀 Features

- Kilograms ↔ Pounds
- Kilometers ↔ Miles
- Liters ↔ Gallons
- Interactive CLI menu
- Input validation and error handling
- Clean modular structure
